Privacy

Token Workshop is local-first and does not include product telemetry.

Data handled by the app

The Figma plugin can read and write local Figma variables, styles, selected layers, and token bindings when you use those features. The local server reads and writes token files in the directory you choose with --dir.

Network access

The plugin connects to http://localhost:9400 so it can communicate with the local Token Workshop server. Git operations only contact remotes you configure yourself.

Storage

The plugin stores preferences, layout settings, recent actions, and drafts in browser local storage or session storage. Token files remain in your chosen local directory unless you explicitly export, copy, or push them elsewhere.

Telemetry

Token Workshop does not send product analytics, telemetry, or usage events. This documentation site does not include analytics scripts.

Privacy questions can be reported through GitHub security advisories.
