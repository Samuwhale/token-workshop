Figma token authoring for local design-token files.

Token Workshop helps designers create, edit, apply, audit, export, and version design tokens from a Figma-native workflow.

Set up Token Workshop View source

Author tokens

Create collections, modes, aliases, generators, and token metadata in a workflow built around Figma's mental model.

Apply in Figma

Publish tokens to local variables and styles, bind tokens to selected layers, and inspect where tokens are used.

Keep files local

The plugin connects to a local server on your machine. Token files stay in the directory you choose.
