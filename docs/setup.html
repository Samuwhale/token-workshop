Setup

Run the local server, then open the Token Workshop plugin in Figma.

Requirements

Install Node.js 20 or newer. Token Workshop uses a local server so your token files can stay on your machine.

Start the server

npx token-workshop --dir ./tokens

The server listens at http://localhost:9400. The directory can be empty; Token Workshop will show an empty workspace until you create or import tokens.

Open the plugin

Install Token Workshop from Figma Community, then run it from a Figma design file. The plugin connects to the local server automatically.

Common fixes

Server offline

Check that the terminal command is still running and that no other process is using port 9400.

Wrong folder

Stop the server and restart it with the token directory you want to edit.

Local network blocked

Allow Figma to connect to localhost when your operating system or browser prompts for local network access.
