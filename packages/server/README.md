# token-workshop

Local server and CLI for Token Workshop, an open-source Figma token authoring tool.

## Usage

```bash
npx token-workshop --dir ./tokens
```

The server listens on `http://localhost:9400` by default and reads/writes token files in the directory passed to `--dir`.
