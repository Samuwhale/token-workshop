# Token Workshop Brand Assets

These files are the plugin icon/logo assets for Token Workshop.

- `token-workshop-logo.png` is the canonical transparent logo source.
- `token-workshop-icon-1024.png`, `token-workshop-icon-512.png`, `token-workshop-icon-256.png`, `token-workshop-icon-128.png`, `token-workshop-icon-64.png`, and `token-workshop-icon-32.png` are square transparent exports for plugin listings, documentation, and small UI contexts.
- `token-workshop-logo-source.png` is the original imagegen chroma-key output kept only for provenance and regeneration reference.

The Figma plugin manifest does not reference these files directly; Figma uses separate listing artwork outside `manifest.json`. The release archive includes the usable transparent logo and icon exports so the packaged plugin has the same brand assets as the repo.
