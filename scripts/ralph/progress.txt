## Codebase Patterns
- Plugin UI is all in `packages/figma-plugin/src/ui/` — App.tsx, hooks/, components/
- `controller.ts` has pre-existing Figma environment type errors (figma global not available in tsc) — ignore these, they are expected
- Build command: `cd packages/figma-plugin && npm run build` (uses vite via build.mjs)
- Typecheck: `npx tsc --noEmit -p packages/figma-plugin/tsconfig.json` — ignore controller.ts errors
- Tailwind CSS variable pattern: `var(--color-figma-*)` for all colors (accent, text, bg, border, etc.)

- Token tree structure: leaf nodes have `$value`, group nodes are plain objects; keys starting with `$` are metadata
