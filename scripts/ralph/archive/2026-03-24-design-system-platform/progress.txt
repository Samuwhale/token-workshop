## Codebase Patterns
- Typecheck plugin: `npx tsc -p packages/figma-plugin/tsconfig.json --noEmit`
- Typecheck server: `npx tsc -p packages/server/tsconfig.json --noEmit`
- PATCH token API: `${serverUrl}/api/tokens/${encodeURIComponent(setName)}/${path.replace(/\./g, '/')}` body `{ $value }`
- Confirmation modals: use in-plugin modal pattern (see ConfirmModal or similar), NOT window.confirm()
- Server endpoints in: packages/server/src/routes/tokens.ts, themes.ts, sync.ts, export.ts
- Plugin UI components in: packages/figma-plugin/src/ui/components/
- Controller messages typed in: packages/figma-plugin/src/plugin/controller.ts
- Toast/snackbar: pushStatus('message') in TokenList.tsx context
- sessionStorage for UI persistence, clientStorage for Figma plugin persistence
