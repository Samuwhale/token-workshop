## 2026-03-27 - ExportPanel From-Figma empty state misleading "Connect to server" message
- The ExportPanel had an early-return guard that showed "Connect to server to export tokens" when not connected, blocking the entire panel — including the "From Figma" tab which doesn't require a server connection (copy as JSON works offline).
- Fix: Removed the early-return guard. Instead, auto-switch to figma-variables mode when disconnected, disable the Platforms tab with a tooltip, and show "(offline)" on the Save to Token Server button with appropriate disabled state.
- Files changed: `packages/figma-plugin/src/ui/components/ExportPanel.tsx` ~L446 (early return removed, useEffect added), ~L468 (platforms tab disabled), ~L999 (save button disabled state)
- **Learnings for future iterations:**
  - When a panel has multiple modes/tabs, don't block the entire panel if only some modes require a dependency (like server connection). Disable the specific modes that need it instead.

## 2026-03-27 - openBindFromProp uses lastIndexOf('.') to derive parent group — breaks for dotted segment names
- Replaced `lastIndexOf('.')` parent derivation with `nodeParentPath(binding, leafName)` using the token's DTCG key.
- Added `$name` field to `TokenMapEntry` interface to carry the DTCG leaf key through the flat token map.
- Wrote `flattenWithNames` helper in `useTokens.ts` that traverses the DTCG group like `flattenTokenGroup` but preserves each leaf's key as `$name`.
- Files changed: `packages/figma-plugin/src/shared/types.ts` ~L140, `packages/figma-plugin/src/ui/hooks/useTokens.ts` ~L7 (helper), ~L83, ~L128 (callsites), `packages/figma-plugin/src/ui/components/SelectionInspector.tsx` ~L10 (import), ~L307 (fix)
- **Learnings for future iterations:**
  - `flattenTokenGroup` from core discards the DTCG key — it only returns `Map<path, token>`. When leaf names are needed downstream, a parallel traversal or extended flatten is required.
  - The `$name` field on `TokenMapEntry` is optional, so existing code that creates entries without it continues to work.
