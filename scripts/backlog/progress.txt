- Files changed: `packages/server/src/routes/tokens.ts` ~L388, `packages/figma-plugin/src/ui/components/TokenList.tsx` ~L1422

  - For group deletes, filter out intra-group dependents (deps whose path is in the `deletedSet`) to avoid false positives when tokens within the same group alias each other.
