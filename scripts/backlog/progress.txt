## 2026-03-27 - ExportPanel From-Figma empty state misleading "Connect to server" message
- The ExportPanel had an early-return guard that showed "Connect to server to export tokens" when not connected, blocking the entire panel — including the "From Figma" tab which doesn't require a server connection (copy as JSON works offline).
- Fix: Removed the early-return guard. Instead, auto-switch to figma-variables mode when disconnected, disable the Platforms tab with a tooltip, and show "(offline)" on the Save to Token Server button with appropriate disabled state.
- Files changed: `packages/figma-plugin/src/ui/components/ExportPanel.tsx` ~L446 (early return removed, useEffect added), ~L468 (platforms tab disabled), ~L999 (save button disabled state)
- **Learnings for future iterations:**
  - When a panel has multiple modes/tabs, don't block the entire panel if only some modes require a dependency (like server connection). Disable the specific modes that need it instead.

## 2026-03-27 - syncBindings with scope 'selection' skips children
- Made `syncBindings` recurse into descendants of selected nodes, matching `remapBindings` behavior. Previously only directly selected nodes were synced; now selecting a frame syncs all bound children too.
- Files changed: `packages/figma-plugin/src/plugin/controller.ts` ~L1203
- **Learnings for future iterations:**
